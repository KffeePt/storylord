# Story Lord Manual

Welcome to **Story Lord**, a powerful CLI tool for managing your story bible, timelines, and world-building data.

> **Full Manual**: See [docs/manual/inputs.md](docs/manual/inputs.md) for detailed controls and [docs/manual/options/](docs/manual/options/) for feature guides.

## Controls

### Global Keys
- **Alt+R**: Restart the CLI immediately.
- **Alt+D**: Toggle Debug Mode overlay.
- **F1 - F4**: Quick Jump to Screens (Dashboard, Generator, Explorer, Sync).
- **Ctrl+C / Q**: Quit (Global safe exit).

### Navigation
- **WASD** or **Arrow Keys**: Navigate menus and lists.
- **Enter / Right / D**: Select item or Enter directory.
- **Left / A**: Go Back or Collapse directory.

## Screens Overview

### 1. Dashboard
The main hub. Access all other modules from here.

### 2. Explorer
A file manager for your story data.
- **Fixed Categories**: Characters, Lore, Canon, Rules, Composition, Storyboard.
- **Storyboard Folder**: Special folder highlighted in **Gold**. Select it to enter the **Timeline Editor**.

### 3. Storyboard (Timeline)
A visual editor for your events.
- Displays events as a vertical chain.
- Shows detailed metadata on the right panel.

### 4. Settings
- **Restart CLI**: Alternative way to reboot.
- **Manual / Documentation**: Browse this manual and other internal docs internally.

## File Structure (`docs/`)
The `docs/` folder contains Git history, archived tasks, and technical specs. You can browse these from the Settings -> Manual screen.
