# Storyboard (Timeline Editor)

The **Storyboard** is a specialized view for managing the sequence of events in your story.

## Usage
1.  Create files in the `Storyboard` category (folder).
2.  Open **Storyboard** from the Dashboard or Explorer (Gold Highlight).
3.  **Visualization**:
    *   Events are shown as a vertical chain: `[ Event A ] -> [ Event B ]`.
    *   The actively selected event is highlighted.
4.  **Details Panel**:
    *   The right side of the screen shows the full content and metadata of the selected event.

## Tips
*   Use this check the flow of your narrative.
*   Ensure every major beat has a corresponding file.
