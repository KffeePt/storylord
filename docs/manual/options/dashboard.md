# Dashboard

The **Dashboard** is the main hub of Story Lord.

## Features
*   **Story Operations**:
    *   **Generator**: Create new story assets (Characters, Lore, etc.) using templates.
    *   **Explorer**: Browse your story files alongside the Manual and Storyboard.
    *   **Storyboard**: Open the Timeline Editor.
    *   **Sync**: Check for integrity issues between your files and the metadata database.
*   **System**:
    *   **Exit**: Close the application.
