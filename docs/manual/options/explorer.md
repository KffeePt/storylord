# Explorer

The **Explorer** is your file manager specialized for Story Bible maintenance.

## Categories
The Explorer enforces a strict structure for consistency:
*   **Storyboard**: (See [Storyboard](storyboard.md))
*   **Characters**: Profiles of your protagonists and antagonists.
*   **Lore**: World-building documents, history, and geography.
*   **Canon**: Established facts and sequence of events.
*   **Rules**: Magic systems, physics, or game rules.
*   **Composition**: Drafts, outlines, and scripts.

## Navigation
*   Use **Enter** or **Right** to open a category.
*   Select a file to view its specific metadata.
