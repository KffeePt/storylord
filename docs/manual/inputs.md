# Inputs & Controls

## Global Shortcuts
*   **Alt+R**: Restart the CLI immediately (Hard Reset).
*   **Alt+D**: Toggle the Debug Overlay.
*   **Ctrl+C**: Force Exit (Safe).

## Navigation
*   **W / Up Arrow**: Move Selection Up.
*   **S / Down Arrow**: Move Selection Down.
*   **A / Left Arrow**: Go Back / Collapse Folder.
*   **D / Right Arrow**: Select / Enter Folder / Expand.

## Quick Jump Keys
*   **F1**: Dashboard
*   **F2**: Generator
*   **F3**: Explorer
*   **F4**: Sync Status
